Downloading an Installer

If you have received ARM Sourcery G++ Lite on a CD, or other physical media, then you do not need to download an installer. You may skip ahead to the section called “Installing ARM Sourcery G++ Lite”.

If you have a ARM Sourcery G++ Lite subscription (or evaluation), then you can log into the Sourcery G++ support site to download your ARM Sourcery G++ Lite toolchain(s). CodeSourcery also makes some toolchains available to the general public from Sourcery G++ web site. These publicly available toolchains do not include all the functionality of CodeSourcery's product releases.

Once you have navigated to the appropriate web site, download the installer that corresponds to your host operating system. For Microsoft Windows systems, the ARM Sourcery G++ Lite installer is provided as an executable, with the .exe extension. For GNU/Linux Solaris systems, ARM Sourcery G++ Lite is provided as a compressed tape archive file, with the .tar.bz2 extension.

On Microsoft Windows systems, save the installer to the desktop. On GNU/Linux and Solaris systems, save the download package in any location that is convenient.
