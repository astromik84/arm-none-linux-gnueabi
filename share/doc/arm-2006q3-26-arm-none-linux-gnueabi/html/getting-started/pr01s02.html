Organization

This document is organized into the following chapters and appendices:

Chapter 1, ARM Sourcery G++ Lite Licenses

This chapter provides information about the software licenses that apply to ARM Sourcery G++ Lite. Read this chapter to understand your legal rights and obligations as a user of ARM Sourcery G++ Lite.

Chapter 2, Sourcery G++ Subscriptions

This chapter provides information about ARM Sourcery G++ Lite subscriptions. CodeSourcery customers with ARM Sourcery G++ Lite subscriptions receive comprehensive support for ARM Sourcery G++ Lite. Read this chapter to find out how to obtain and use a ARM Sourcery G++ Lite subscription.

Chapter 3, ARM Sourcery G++ Lite for ARM GNU/Linux

This chapter provides information about this release of ARM Sourcery G++ Lite including any special installation instructions, recent improvements, or other similar information. You should read this chapter before building applications with ARM Sourcery G++ Lite.

Chapter 4, Installation and Configuration

This chapter describes how to download, install and configure ARM Sourcery G++ Lite. This section describes the available installation options and explains how to set up your environment so that you can build applications.

Chapter 5, Using ARM Sourcery G++ Lite with Eclipse

This chapter explains how to install the ARM Sourcery G++ Lite Eclipse plugin so that you can use ARM Sourcery G++ Lite with the Eclipse integrated development environment.

Chapter 6, Using Sourcery G++ from the Command Line

This chapter explains how to build applications with ARM Sourcery G++ Lite using the command line. In the process of reading this chapter, you will build a simple application that you can use as a model for your own programs.

Appendix A, GNU General Public License

This appendix contains the full text of the GNU General Public License, the license that applies to certain components of ARM Sourcery G++ Lite.
