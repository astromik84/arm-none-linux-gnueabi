Chapter 5. Using ARM Sourcery G++ Lite with Eclipse

Abstract

This chapter explains how to use the Eclipse IDE to build a C or C++ application with ARM Sourcery G++ Lite. This chapter assumes you have installed ARM Sourcery G++ Lite as described in Chapter 4, Installation and Configuration. If you would prefer to use the command line to build your applications, you may refer to Chapter 6, Using Sourcery G++ from the Command Line instead.

The Eclipse IDE is not included in Sourcery G++ Lite. If you are using Sourcery G++ Lite, you can skip this chapter.
