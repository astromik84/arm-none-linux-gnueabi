Abstract

This guide explains how to install and build applications with ARM Sourcery G++ Lite, CodeSourcery's customized, validated, and supported version of the GNU Toolchain. ARM Sourcery G++ Lite includes everything you need for application development, including C and C++ compilers, assemblers, linkers, and libraries.

When you have finished reading this guide, you will know how to use ARM Sourcery G++ Lite from the command-line and from the Eclipse IDE.
