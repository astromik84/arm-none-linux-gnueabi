System Requirements

Host Operating System Requirements

ARM Sourcery G++ Lite supports the following host operating systems:

  • Microsoft Windows NT 4, Windows 2000, and Windows XP systems using IA32, AMD64, and EM64T processors.

  • GNU/Linux systems using the IA32, AMD64, or EM64T processors, including Debian 3.0 (and later), Red Hat Enterprise Linux 3 (and later), SuSE Enterprise Linux 8 (and later).

  • Solaris 2.8 (and later) systems using SPARC processors.

Not all combinations of host and target systems are available. Therefore, ARM Sourcery G++ Lite for your target system may not be available on all of the above host systems.

Host Hardware Requirements

In order to install and use ARM Sourcery G++ Lite, you must have:

  • At least 200MB of free disk space.

  • At least 64MB of available memory.

Target System Requirements

See Chapter 3, ARM Sourcery G++ Lite for ARM GNU/Linux for requirements that apply to the target system.
